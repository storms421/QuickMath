## Quick Math
  This is a browser based math game built as a personal learning project using HTML, CSS, and JavaScript. The idea was to work alongside Claude while developing this project, having Claude be used as both a tutor (for learning) and a partner (for bouncing off ideas).

## How to Play
  Quick Math gives you 60 seconds to answer as many math problems as possible. Each problem has four answer choices to pick from. If the answer chosen is incorrect, the player is hit with a 3 second penalty before continuing on to the next problem. The final score is the total number of correct answers when the timer hits zero. Players have the chance to take a spot on the default leaderboard in which they can place their own initials for their potential new placement.

## Problem Types
  Players will have 4 different types of problems they can face:
  
    1. Addition: Numbers 1-15
    2. Subtraction: Numbers 1-15 (Possibility of Negative Answers)
    3. Multiplication: Numbers 1-12
    4. Division: Numbers 1-12 (Always Produce a Whole Number)

## Achievements
  This game has a fun aspect of earning achievements! Currently, there are 9 achievements that can be earned. It is up to you to figure them out. Have fun!

## Getting Started
  1. Download the project folder
  2. Open QuickMath.html in a browser of your choosing (Chrome, Edge, and Firefox for Example)
  3. No installation or internet connection needed!

## Resetting Game Data
  The game data is saved locally in your browser. To reset:

  ### Clear Everything:
    1. Press F12 to open DevTools
    2. Click the Application tab
    3. Click Storage in the left panel
    4. Click Clear site data

  ### Clear Only Leaderboard or Achievements:
    1. Press F12 to open DevTools
    2. Click the Application tab
    3. Expand Local Storage in the left panel and click 'file:///'
    4. Click the entry you want to remove (leaderboard or achievements)
    5. Press the ‘X’ button at the top to delete it
